# MOAP Cloud v2 数据核对

本项目的数据口径按上传的 `MOAP_Official_v10.1_LTS_Official.xlsx` 核对：

- `Players`：5 名玩家
- `Matches`：65 场比赛
- `Results`：325 条逐人结果（每场包含 5 名成员，缺席者以空分记录）
- `Statistics_Summary`：生涯积分、场均、正分率、MVP、四人局/五人局场均
- `BGR_Dashboard_v26`：50–59 / 60–69 / 70–79 / 80–89 / 90–99 / 100+，权重 1 / 2 / 3 / 5 / 8 / 12
- `Honor_Center` / `Award_Catalog`：荣誉档案
- `AE027_GOAT_Engine`：GOAT = 荣誉积分 + 生涯 MVP + 赛季冠军 × 10
